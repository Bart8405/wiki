# Concepts Index

*Global cross-project index of highlighted concepts. Aggregated from per-project review files (e.g. [IRO_review](../projects/finals-y2-s2/wiki/synthesis/IRO_review.md), [PoS_review](../projects/finals-y2-s2/wiki/synthesis/PoS_review.md)).*

**How to use:**
- While reading any wiki note in Obsidian, highlight unfamiliar terms with `==term==` and add questions with `%%question%%`.
- Run `/extract-highlights` to (a) update the per-project review file with full definitions, and (b) refresh this global index with a one-line entry pointing to the per-project file.
- Click a row to jump straight to the enriched definition + your original highlight.
