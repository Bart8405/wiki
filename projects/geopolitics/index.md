# Geopolitics — Project Index

*Cross-cutting reference project for countries, institutions, and power blocs. When a concept in any other project relates to a specific geopolitical actor, the actor gets an entry here with backlinks to where it was discussed.*

## Why this project exists

When studying IR, GPE, or anything related to political economy, the same actors keep recurring (ASEAN, China, the EU, etc.) but the concepts attached to them are scattered across different course summaries and projects. This project gathers them in one place so that clicking on "ASEAN" gives you the full picture of every concept, lecture, and reading where it came up.

**Naming convention:** entity pages use the actor's common name (lowercase, hyphen-separated): `asean.md`, `china.md`, `european-union.md`, `brics.md`.
