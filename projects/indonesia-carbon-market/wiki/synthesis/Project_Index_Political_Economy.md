# Project Index — Political Economy

*A project-by-project index of political-economy flags in the Indonesian voluntary carbon market dataset. Same information as [[Political_Economy_Notes]], but flipped from theme-first to project-first so you can quickly ask "is project X suspicious?" and find what we know.*

---

## How to use this file

1. **Scroll** to the project ID you care about (they're in numerical-then-alphabetical order within registry).
2. **Each entry** lists relevant flags, named people, and a `→` cross-reference back to [[Political_Economy_Notes]] for the deeper context.
3. **"No PE flags currently identified"** means nothing has been documented in the wiki yet — *absence is not proof of cleanliness*. Update by adding a bullet point and a back-link.
4. **Update workflow:** whenever you add a flag in [[Political_Economy_Notes]], also add a one-line entry here under the relevant project ID, with a `→` link to the deeper note. Keep this index thin (1–6 bullets per project) and the PE Notes file rich.
